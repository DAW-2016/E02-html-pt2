HTML: introdução e estrutura básica

HTML significa: HyperText Markup Language. HTML é uma linguagem voltada para marcação/estruturação de documentos da web. Existem vários recursos para cada tipo de coisa que se deseja criar em HTML. Os recursos para a demarcação do hipertexto nós chamamos de tags. As tags funcionam como funções pre-definidas para estruturar nossa pagina na web de forma organizada: título, cabeçalho, corpo do texto, barra de pesquisas, listas. Toda tag é composta de nome, atributos e valor, como no exemplo: Clique aqui (Clique com o botão direito sobre o link e em seguida clique em inspecionar elemento para identificar a sintaxe da tag).

Elemento de containers

  • Head: Cabeçalho onde são armazenadas informações da página.

    •     Exemplo: title, css, meta...

  • Body: O corpo do HTML. Dentro dele se encontra todo o conteúdo da página.

    •     Exemplo: Tags como p, h1, h2, ol, ul... Todas são usadas dentro do "body".

  • Div: Tag usada para agrupar elementos em um espaço determinado

    •     Exemplo: Veja o exemplo do uso de divs abaixo.

  • Header e Footer: divs usadas em ocasiões especiais que preenchem o cabeçalho e o rodapé da página, respectivamente

    •     O título desta página (Manual de referência para HTML) poderia estar entre a tag header.

        No final da página temos o rodapé feito com a tag footer.
